# 🇨🇴 DANE EEVV — Datos de Nacimientos Colombia (2020-2023)

Repositorio de microdatos oficiales de nacimientos del DANE (Estadísticas Vitales - EEVV),
utilizados para el proyecto de predicción de nacimientos por departamento con Machine Learning.

## Contenido

| Archivo | Año | Registros | Fuente oficial |
|---|---|---|---|
| nac2020.csv | 2020 | 629,402 | [DANE EEVV 2020](https://microdatos.dane.gov.co/index.php/catalog/732/get-microdata) |
| nac2021.csv | 2021 | 616,914 | [DANE EEVV 2021](https://microdatos.dane.gov.co/index.php/catalog/775/get-microdata) |
| nac2022.csv | 2022 | 573,625 | [DANE EEVV 2022](https://microdatos.dane.gov.co/index.php/catalog/807/get-microdata) |
| nac2023.csv | 2023 | 515,549 | [DANE EEVV 2023](https://microdatos.dane.gov.co/index.php/catalog/876/get-microdata) |

## Proyecto

Este repositorio es la fuente de datos del siguiente cuaderno de Google Colab:

🔗 [Predicción de Nacimientos en Colombia 2024](https://colab.research.google.com/drive/1fORkcwMrJFrOYnHNNRAozohdwjZFHUfb#scrollTo=66xhqDTZ93GS)

El cuaderno implementa un modelo de Machine Learning (Regresión Lineal)
para predecir el número de nacimientos por departamento en Colombia para el año 2024,
entrenado con los datos históricos 2020-2023.
